# sample-ruby-project

## Install

```
bundle install --path vendor/bundle
```

## Rake Tasks

```
$ bundle exec rake -vT
rake sample:crawler[url]      # Crawler
rake sample:day_time[time]    # DayTime
rake sample:fizz_buzz[count]  # FizzBuzz
rake sample:space             # Space
```

## Execution

```
bundle exec rake abc:xyz
```
